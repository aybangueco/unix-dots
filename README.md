# unix-dots

My personal dotfile configuration that works with Linux and MacOS.
(I miss tiling window manager)

## Prerequisites

- nvm
- gobrew
- starship
- zsh
- a package manager (pacman, homebrew watevahh works)
- neovim
- dotsync (my personal util)
- zed (sometimes i use this)
- nvim (my prefer editor as of now because it doesn't feel heavy)
- ghostty
